Первым запускать файл Wheather.exe. Он начнет эмулировать суточные изменения 
атмосферного давления и температуры в соответствии с одним определенным летним днем.

Вторым запустить Kotel.exe. Он будет эмулировать работу котла в соответствии с
законом охлаждения Ньютона, а также законами термодинамики для жидкостей и газов.
Этот файл сразу работать не начнет. Вместе с ним нужно запустить Regulator.exe.
Он будет выполнять роль регулятора, получая данные о температуре котла, выводя их на экран и, 
принимая решения по управлению.

Создаваемые в процессе работы файлы это своеобразный глобальный массив, в который программы помещают свои переменные.

Wheather.exe помещает значение атмосферного давления в мм.рт.ст. в файл externalPressure.bnr, а 
значение температуры окружающей среды в файл externalTemperature.bnr.

Kotel.exe помещает вычисленное значение внутренней температуры в файл insideTemperature.bnr и 
читает значение открытия газа из файла gasPosition.bnr.

Regulator.exe помещает значение открытия или закрытия газового клапана в файл gasPosition.bnr 
и читает значение внутренней температуры из файла insideTemperature.bnr.
